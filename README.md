# 📌 Monthly Excel -- Cafe URL 기반 업무 자동화 프로그램

카페 URL만 입력하면 **크롤링 → 데이터 정제 → Excel 템플릿 자동
작성**까지\
전 과정을 자동화하는 **윈도우 데스크탑 업무 자동화 도구**입니다.

------------------------------------------------------------------------

## 🚀 왜 만들었는가?

여자친구 회사에서 매일 반복되는 아래 작업을 자동화하기 위해
제작되었습니다:

-   여러 네이버 카페 게시글 확인\
-   조회수·댓글수 수집\
-   텍스트 정제 및 분류\
-   Excel 템플릿 수동 기입\
-   수십 번 반복되는 비효율적인 업무

이 프로그램은 전체 과정을 버튼 한 번으로 자동 실행하여\
**업무 시간을 90% 이상 단축**하도록 설계되었습니다.

------------------------------------------------------------------------

## 🛠 주요 기능

### 🔗 1. URL 기반 자동 크롤링

-   네이버 카페 게시물 URL만 입력하면\
    Playwright를 이용해 자동 접속 → 제목/내용/조회수/댓글/작성일 수집

### 📑 2. 조회수·댓글수 기준 정렬 기능

-   크롤링된 데이터 자동 정렬\
-   높은 조회수/최근 날짜 순으로 자동 정렬 가능

### 📄 3. Excel 템플릿 자동 기입

-   ClosedXML 기반 Excel 템플릿 자동 작성\
-   텍스트 파싱 및 키워드 분리 기능 포함\
-   각 셀에 자동 매핑

### 📤 4. 템플릿 다운로드 / 업로드 기능

-   기본 제공 업로드 템플릿 다운로드\
-   사용자가 원하는 템플릿으로 교체 가능

### ⚡ 5. 빠른 처리 속도

-   대량 게시글을 수초 내 처리\
-   한 번의 클릭으로 전체 자동 수행

## 🖥 사용 방법

1.  프로그램 실행\
2.  템플릿 다운로드 (선택)\
3.  네이버 카페 URL 입력\
4.  "크롤링 시작" 클릭\
5.  자동 정렬 + 파싱 + Excel 기입\
6.  결과 Excel 저장
